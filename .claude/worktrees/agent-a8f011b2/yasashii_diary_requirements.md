# 要件定義 - やさしい交換日記

## 1. アプリ概要

### コンセプト
夫婦やパートナー向けの交換日記アプリ。書いた日記は翌月1日に自動的に相手も見れるようになる。書いた直後の感情的な内容を見せたくない場合は、月末までに削除・編集できる仕組み。

### ターゲットユーザー
- 夫婦・カップル（2人専用）
- 日々の感情やモヤモヤを記録したい人
- パートナーとの関係を改善したい人

---

## 2. 主要な機能仕様

### 2.1 基本ルール

#### 日記の公開フロー（月ごと一括公開）
1. **書く**：カレンダーの日付をタップして日記を書く
2. **見返す・編集・削除**：書いた本人はいつでも当月の日記を見返せる、編集できる、削除できる
3. **自動公開**：翌月1日に前月分が一括公開（相手も見れるようになる）
4. **ロック**：公開後は編集・削除不可

**例：**
- 1月15日に書いた日記 → 2月1日に公開
- 1月31日に書いた日記 → 2月1日に公開（翌日！）
- 見せたくない場合は1月31日までに削除

#### テキスト仕様
- プレーンテキストのみ（写真添付なし）
- 1日1つの日記のみ
- 1日あたり400文字まで
- リアルタイムで「あと◯◯文字書けます」表示

#### ユーザー関係
- 2人専用（グループ機能なし）
- どちらか一方が招待する形式

---

### 2.2 日記作成フロー

#### 日記を書く
1. カレンダーの日付をタップ
2. 日記作成画面が開く
3. テキストを入力（400文字まで）
4. 保存

#### 操作ルール
- **今月の日付（日記なし）**：タップで新規作成
- **今月の日付（日記あり）**：タップで編集・削除画面へ
- **過去月・未来月**：操作不可（カレンダーは今月のみ）

---

### 2.3 通知

以下のタイミングで通知：
- **月末の数日前**：「まもなく◯月の日記が公開されます」
- **相手がスタンプをつけた**

---

## 3. 画面構成

### 3.1 画面遷移の全体像

```
日記帳ビュー（トップ）←→ カレンダービュー（今月のみ）
     ↓
日記帳ビュー（月別詳細）
```

- **左右スワイプ**で日記帳ビュー ←→ カレンダービューを切り替え
- デフォルトはカレンダービュー

---

### 3.2 カレンダービュー（今月のみ）

#### レイアウト
- Googleカレンダー風のUI
- **今月のカレンダー固定**（月送り・月戻しなし）
- 月ヘッダーに「📖」ボタン → 日記帳ビュー（トップ）へ
- **右スワイプ** → 日記帳ビュー（トップ）へ

#### 日付の表示ルール

**未公開日記（今月）：**
- 自分のカラーの四角形のみ
- タップ → 編集・削除画面

**日記がない日付：**
- タップ → 新規作成画面

**過去月・公開済み：**
- カレンダーには表示なし（日記帳ビューで見る）

#### 操作
- 日付タップで日記作成・編集
- 右スワイプで日記帳ビューへ

---

### 3.3 日記帳ビュー（トップ）

#### レイアウト
```
┌─────────────────────┐
│  📖 日記帳          │
├─────────────────────┤
│                     │
│  📔 2026年1月       │
│  15件の日記         │
│                     │
├─────────────────────┤
│  📔 2025年12月      │
│  23件の日記         │
│                     │
├─────────────────────┤
│  📔 2025年11月      │
│  18件の日記         │
│                     │
└─────────────────────┘
```

#### 機能
- 過去月を月ごとのカードで表示
- カードに絵文字「📔」と日記件数
- カードタップ → その月の日記詳細へ
- **左スワイプ** → カレンダービューへ

---

### 3.4 日記帳ビュー（月別詳細）

#### レイアウト
- 紙の日記帳のようなデザイン
- 選択した月の日記を時系列表示（日付昇順）
- 自分と相手の日記が混ざって表示

#### 日記の表示形式
```
━━━ 1月15日 ━━━

[Aさん 10:00]
朝、イライラした。
洗い物が溜まっていて...

❤️ Bさん

━━━ 1月16日 ━━━

[Bさん 09:00]
朝ごはんが美味しかった。

😊 Aさん

[Aさん 22:00]
今日は疲れた...

👍 Bさん
```

#### スタンプ機能
- 日記の下に「スタンプ」ボタン
- 絵文字ピッカーが開く
- スタンプ表示：「❤️ Bさん」のように日記の下に表示

#### 操作
- 編集・削除不可（公開済み）
- スタンプのみ可能
- 戻るボタンで日記帳トップへ

---

### 3.5 日記作成・編集画面（モーダル）

#### 新規作成時
```
┌─────────────────────┐
│ 1月15日の日記を書く  │
│                     │
│ ┌─────────────────┐ │
│ │                 │ │
│ │ テキストエリア    │ │
│ │                 │ │
│ └─────────────────┘ │
│                     │
│ あと400文字書けます  │
│                     │
│      [保存]          │
└─────────────────────┘
```

#### 編集時
```
┌─────────────────────┐
│ 1月15日の日記       │
│                     │
│ ┌─────────────────┐ │
│ │ 朝、イライラした  │ │
│ │ 洗い物が...      │ │
│ └─────────────────┘ │
│                     │
│ 🔒 2月1日に公開     │
│ あと250文字書けます  │
│                     │
│  [編集]  [削除]     │
└─────────────────────┘
```

#### 操作
- **新規作成**：テキスト入力 → 保存
- **編集**：既存テキストを編集可能
- **削除**：物理削除（確認ダイアログあり）
- **月を跨いだら**：編集・削除ボタン非表示（ロック）

- 作成日時

- 日記エントリID

- 絵文字
- 作成日時

## 9. ビジネスロジック詳細

### 月ごと一括公開の仕組み
```
1月中に書いた日記:
- 1月1日 10:00に作成 → 2月1日 00:00公開
- 1月15日 14:00に作成 → 2月1日 00:00公開
- 1月31日 23:59に作成 → 2月1日 00:00公開（翌日！）

編集・削除可能期間: 
- 1月1日 - 1月31日 23:59まで

ロック:
- 2月1日 00:00以降は編集・削除不可
```

### 通知タイミング
```
1月の日記 → 2月1日公開

通知: 1月28日（3日前）
「まもなく1月の日記が公開されます。
見直したい日記があれば今のうちに編集・削除できます。」
```

### 削除の扱い
- 物理削除（データベースから完全に削除）
- 削除確認ダイアログ：「この日記を削除しますか？削除すると復元できません。」
- 削除後、カレンダーから完全に消える（相手にも痕跡なし）

### 1日1つの日記制限
- 同じ日に複数の日記は作成不可
- 既に日記がある日付をタップ → 編集画面が開く
- 書き直したい場合は編集で対応

### 日記帳の月カード表示ルール
- 公開済み月のみ表示
- 例：今が2026年3月 → 2026年2月、2026年1月、2025年12月... と表示
- 今月（2026年3月）はカレンダービューで管理
